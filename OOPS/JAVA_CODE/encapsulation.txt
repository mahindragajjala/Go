In Java, encapsulation is achieved by:
Making class variables private
Providing public getter and setter methods to access and modify them.


public class Person {
    // Private data members - hidden from outside
    private String name;
    private int age;

    // Public methods to access private members
    public String getName() {
        return name;
    }

    public void setName(String name) {
        if (name.length() > 0) {
            this.name = name;
        }
    }

    public int getAge() {
        return age;
    }

    public void setAge(int age) {
        if (age > 0) {
            this.age = age;
        }
    }
}

public class Main {
    public static void main(String[] args) {
        Person p = new Person();
        p.setName("Alice");
        p.setAge(25);

        System.out.println(p.getName()); // Alice
        System.out.println(p.getAge());  // 25
    }
}
